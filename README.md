## üõ†Ô∏è Getting Started

1. **Clone the repository:**

   ```bash
   git clone https://github.com/PhantomHerald/Twitch_Clone.git
   cd Twitch_Clone
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Run the app:**

   ```bash
   npx expo start
   ```

   Scan the QR code with the **Expo Go** app or run on an emulator.
